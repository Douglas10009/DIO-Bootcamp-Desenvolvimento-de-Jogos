Object Fit

Object-fit define se o objeto deve se alterar para caber no conteiner


Object-fit: fill; (Default)

Preenche todo o conteiner


Object-fit: contain;

Cabe de acordo com o tamanho da imagem


Object-fit: cover;

A imagem vai preencher o conteiner, mas diferente do fill, ele não vai distorcer pois irá cortar as laterais


Object-fit: none;

Não aplica object-fit e deixa a imagem no tamanho normal


Object-fit: scale-down;

Escolhe entre o contein, none ou fill para escolher a menor imagem


Object Position

Define como uma média deve ser posicionado, horizontalmente/verticalmente

object-position: [x] [y]
